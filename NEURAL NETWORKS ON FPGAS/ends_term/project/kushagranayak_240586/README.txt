## 📌 Project Overview
This project demonstrates the end-to-end development, training, and hardware deployment of an Artificial Neural Network (ANN) on an FPGA. The model was trained in Python using PyTorch to classify the Iris dataset and synthesized into custom Verilog hardware using Xilinx Vivado.

The network architecture is a **4-8-3 Multi-Layer Perceptron**:
* **Input Layer:** 4 nodes (sepal/petal dimensions)
* **Hidden Layer:** 8 neurons
* **Output Layer:** 3 nodes (Iris species classifications)

To bridge the gap between software and hardware, the floating-point weights and biases generated by PyTorch were converted into **Q8 Fixed-Point representation** (16-bit Two's Complement) and loaded into the FPGA's memory via `.mem` files.

## 🚀 Performance Metrics & Hardware Utilization

The design was successfully synthesized and implemented for the **Artix-7** architecture. The timing constraints were met with significant margin, proving a highly optimized hardware datapath.

| Metric | Result |
| :--- | :--- |
| **Target Board** | Nexys A7-100T (`xc7a100tcsg324-1`) |
| **Clock Constraint** | 100 MHz (10.000 ns period) |
| **Worst Negative Slack (WNS)** | **8.866 ns** |
| **Total Negative Slack (TNS)** | **0.000 ns** (Timing Passed) |
| **Logic Utilization (LUT)** | **1%** |
| **Status** | Bitstream Generated Successfully |

*Note: A WNS of 8.866ns indicates the logic propagation delay is only ~1.134ns, allowing for massive potential scaling or much faster clock frequencies.*
